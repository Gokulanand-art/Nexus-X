# Nexus X — Offline AI Coding Assistant

100% local. No internet after install. Powered by Phi-3 Mini via Ollama.

## Install

### Linux / macOS
curl -fsSL https://raw.githubusercontent.com/Gokulanand-art/nexus-x/main/install.sh | bash

### Windows (PowerShell as Admin)
irm https://raw.githubusercontent.com/Gokulanand-art/nexus-x/main/install.ps1 | iex

## Run
nexus

## Features
- Agent loop — think, act, observe, repeat
- Tools — read/write files, run shell commands, search code
- Vector memory — remembers every conversation (ChromaDB)
- Mistake learning — never repeats the same error
- Autonomous mode — /run <goal> works alone
- 100% offline after setup
